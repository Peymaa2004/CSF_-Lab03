                     Brief explanation of each task 
            #Condition Statement
1.Check whether a number is positive, negative, or zero. The program requires you to enter a number which the program uses together with conditional statements (if, elif, else) to determine its value as Positive, when it exceeds 0 as Negative, when it is less than 0 and as Zero when it equals 0. 

2.Check whether a number is even or odd. The task requires the use of the modulus operator (%) to check whether a number is divisible by 2. The system identifies even numbers through the condition of number % 2 == 0. All other numbers become classified as odd numbers. 

3.Find the largest of two numbers The program uses first number comparison against second number through if-else statement to determine which number is larger. The first number becomes larger than the second number in the case that first number exceeds second number. The second number becomes larger than the first number in all other cases. 

4.Find the largest of three numbers. The process requires you to compare three numbers through multiple conditions. You must first identify the number which exceeds both of the other numbers. The largest number will be determined through if-elif-else statements. 

5.Take marks as input and display grade. The system uses conditions to determine grades based on marks: Agrade requires students to achieve 80 marks or beyond Bgrade needs students to reach between 60 to 79 marks Cgrade needs students to reach between 40 to 59 marks The system recognizes students as failures who score below 40. 

6.Menu-driven program (add, subtract, multiply, divide). A menu exists which enables users to choose their desired operation between: Add Subtract Multiply Divide The system requires users to enter two numbers before it will execute their selected operation through if-elif-else structures.
